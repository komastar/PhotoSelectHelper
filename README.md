# 파일들을 받았을때
1. `Select` 버튼 클릭
2. 셀렉된 파일들이 있는 폴더 선택
3. `Target` 버튼 클릭
4. raw 파일 등 원본 파일이 있는 경로 선택
5. `Move` or `Copy` 버튼 클릭
6. 완료

# 번호 목록을 받았을때
1. 아무 경로에 `selected.txt` 파일 생성
2. 셀렉된 번호 목록을 `selected.txt` 파일에 기록(1줄에 번호 1개만)
3. `Target` 버튼 클릭
4. raw 파일 등 원본 파일이 있는 경로 선택
5. `Move` or `Copy` 버튼 클릭
6. 완료
